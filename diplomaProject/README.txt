Web часть дипломного проекта. Взаимодействует с пользователем посредством http и аппаратной частью посредством socket. Рeализована возможность оповещения через email, просмотра архива видеофайлов, задания прав доступа. 
Идея дипломного проекта - создание робототехнического комплекса, способного выполнять функции охраны помещения.
